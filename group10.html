【SDGs4】「Oh!Show!繪課室」文藻小學堂 － CODE CODE CODE!

我們是一群熱愛服務的學生，在學習過程曾積極參與教育部數位學伴活動，由於偏鄉弱勢族群教育問題日益受到重視，因此我們以聯合國永續發展【SDGs4】作為目標，和貫徹 108 新課程綱要核心素養之「科技資訊與媒體素養」的理念為宗旨，倡導在學習資源不足地區及學習弱勢學童，亦應獲得公平學習機會及學習環境，在此理念下，本專題計畫針對程式設計的教學需求，進行學習系統的開發，幫助學童可以快速地進入程式設計的領域。該系統將提供一個方便師生使用的平台，老師可以上傳教學內容，學生可以透過平台做課程學習，系統完成後，將會優先提供數位學伴做使用，未來會進一步與偏鄉小學進行合作。本學習系統主要特色：
1. 循序漸進的學習方式：為了確定學童對於課程內容完全理解，致力於更好的 學習方式，穩紮穩打，循序漸進，學童要進入到下一個教材之前，必須完成 該單元練習，確保有一定的熟悉程度，才能進行到下一個教材學習。 2. 即時了解學生學習進度：老師可以透過後台了解學生學習活動，幫助老師掌 握課程學生的學習狀況以及所完成相關進度。 3. 討論區與教材單元緊密結合：每個單元都有個別討論區，使用者可以根據題 目做提問，針對教材內容及練習題目提出相關看法等進行留言，亦可對其他 人的留言來表示贊同或不贊同，老師也可以置頂優秀的留言，供學童參考
